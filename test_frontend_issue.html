Common Issues & Solutions:

  1. Port Confusion: Frontend runs on port 3001, Backend runs on port 8000
  2. Authentication Required: /projects route requires login. Try http://localhost:3001/login
  3. CORS Issues: Backend must allow requests from frontend origin (http://localhost:3001)
  4. API URL Configuration: Frontend .env should have REACT_APP_API_URL=http://127.0.0.1:8000/api/v1
